Cursus Frontend Angular

Un parcours de formation pour apprendre à créer des applications avec Angular.

  • #1 Introduction à NodeJS et NPM

    Compétences à acquérir :

    • Ecrire un programme NodeJS avec ES5
    • Créer un projet NPM
    • Gérer les dépendances via NPM
    • Piloter les scripts via NPM

  • #2 ECMAScript Next

    Compétences à acquérir :

    • Nouvelles syntaxes ES2015+
    • Comprendre les modules ES2015
    • Comprendre la notion de promesse

  • #5 TypeScript

    Compétences à acquérir :

    • Comprendre ce qu'est TypeScript
    • Construire des structures typées avec TypeScript

  • #4 Webpack

    Compétences à acquérir :

    • Comprendre l'intérêt de Webpack
    • Mettre en oeuvre les modules ES2015 avec Webpack

  • #5 Angular

    Compétences à acquérir :

    • Créer une application Angular
    • Créer un composant graphique
    • Echanger des données avec un serveur
    • Gérer la communication entre composants
